Política de Privacidade — Wallpapers App

Última atualização: 06 de julho de 2026

Esta Política de Privacidade descreve como o aplicativo Wallpapers App ("o App"), desenvolvido por NewCore Technologies, coleta, usa e protege as informações dos usuários.

1. Dados coletados

O App foi projetado para minimizar a coleta de dados pessoais. As seguintes informações podem ser processadas:

  • Preferências e favoritos: imagens marcadas como favoritas, categorias visualizadas e configurações do usuário são armazenadas localmente no dispositivo usando o banco de dados Room, e não são enviadas a servidores externos.
  • Identificadores de publicidade: conforme detalhado na seção 3 (AdMob).
  • Dados técnicos de uso: informações agregadas e anônimas de desempenho e falhas podem ser coletadas para fins de estabilidade do App.

O App não requer criação de conta, login ou coleta de dados pessoais identificáveis (nome, e-mail, telefone) para funcionar.

2. Conteúdo de terceiros — API do Pixabay

O App utiliza a API do Pixabay para exibir imagens de papéis de parede. Ao carregar imagens:

  • Requisições são feitas diretamente aos servidores do Pixabay para buscar e exibir conteúdo de imagem.
  • O Pixabay pode registrar metadados técnicos da requisição (como endereço IP) de acordo com sua própria política de privacidade e termos de uso, que recomendamos a leitura.
  • O App não associa buscas ou downloads de imagens a uma identidade pessoal do usuário.

3. Publicidade — Google AdMob

O App exibe anúncios por meio do Google AdMob. O AdMob pode coletar e processar:

  • Identificador de publicidade do dispositivo (Advertising ID / GAID);
  • Dados de interação com anúncios (impressões, cliques);
  • Informações técnicas do dispositivo e da rede.

Esses dados são regidos pela Política de Privacidade do Google e pelas políticas do AdMob para desenvolvedores. O usuário pode gerenciar preferências de anúncios personalizados diretamente nas configurações do Android (Google > Anúncios).

4. Armazenamento local (Room)

Preferências, cache de imagens e configurações são armazenados localmente no dispositivo usando o Room (SQLite). Esses dados não são sincronizados com nenhum servidor e são removidos automaticamente ao desinstalar o App.

5. Compartilhamento de dados

O App não vende nem compartilha dados pessoais com terceiros, exceto os dados técnicos necessários ao funcionamento do AdMob e da API do Pixabay, descritos acima.

6. Público infantil

O App não é direcionado especificamente a crianças menores de 13 anos e não solicita intencionalmente dados pessoais desse público. Caso o App venha a ser classificado sob políticas de conteúdo familiar, os anúncios exibidos serão configurados para conformidade com as diretrizes da Google Play Families Policy.

7. Segurança

Adotamos práticas razoáveis para proteger as informações processadas pelo App, incluindo o uso de conexões HTTPS nas requisições à API do Pixabay.

8. Alterações nesta política

Esta política pode ser atualizada periodicamente. Alterações significativas serão refletidas na data de "Última atualização" no topo desta página.
